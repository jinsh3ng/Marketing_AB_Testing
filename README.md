## Setup

1. Clone the repository
```bash
git clone https://github.com/jinsh3ng/AB_Testing.git
cd AB_Testing
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the notebook
```bash
jupyter notebook main.ipynb
```
